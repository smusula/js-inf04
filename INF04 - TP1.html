Séance TP1 : Structuration de l'application

Vous allez travailler sur une application de mémorisation des contacts, de type répertoire. Cette application gèrera en JavaScript une liste d'objets "Contact". Elle permettra de renseigner un nouveau contact, d'en afficher ou d'en rechercher un et d'afficher la liste des contacts connus.

Dans cette question, vous allez définir la structure de votre application. Cette application sera contenue dans une seule page Web, mais sera constituée de plusieurs vues :

  • Une page d'accueil, contenant un menu permettant d'accéder aux autres vues
  • L'affichage de la liste des contacts déjà enregistrés
  • L'affichage des détails d'un contact particulier
  • Un formulaire de recherche permettant d'entrer du texte et de lancer une recherche dans la liste
  • Un formulaire d'ajout de contact qui permettra d'indiquer les nom, prénom, adresse, adresse mail, numéro de téléphone et date de naissance d'un contact.

Création des vues

Commencez par écrire la structure générale de la page. Déterminez la balise HTML5 qui contiendra les différentes vues et dupliquez-la autant de fois que nécessaire. Choisissez le mécanisme qui permettra de différencier les vues entre elles pour n'en afficher qu'une.

Toutes les vues autres que la page d'accueil devront posséder un bouton permettant de revenir sur la page d'accueil de l'application.

Styles d'affichage

Définissez deux classes de style CSS : la première permettra d'afficher le contenu d'une vue et la seconde de le cacher.

Mécanisme de changement de vue

Créez une fonction JavaScript qui cache la vue actuellement affichée et affiche celle passée en paramètre. Appelez cette fonction avec les bons paramètres depuis les éléments de la page dédiés à la navigation entre les vues. Testez.

Mise en place du métier

Rendez votre application fonctionnelle :

  1. Réalisez le formulaire d'ajout d'un contact, avec les éléments input appropriés
  2. Mettez en place le script correspondant :
    1. Créez une variable globale dans votre application qui stockera en mémoire la liste de contacts
    2. Créez un objet JavaScript qui définit la notion de contact. Pour cela, vous pouvez utiliser la notion de prototype en JS (exemple ici)
    3. Mettez en place le script qui crée un nouveau contact à l'appui sur le bouton de validation et l'ajoute à la liste
  3. Réalisez les vues (et les cripts correspondants) pour l'affichage de la liste et des détails d'un contact
  4. Implémentez un moteur de recherche basique (recherche de chaînes) à l'aide d'un formulaire de recherche et de l'affichage de ses résultats

Ajout de styles

Utilisez vos connaissances en CSS pour styler les vues à votre guise. Réalisez deux feuilles de style : l'une pour affichage sur "grand" écran et l'autre sur mobile. Pour l'instant, appliquez l'uine ou l'autre de ces feuilles, par un appel "en dur" dans le code source de la page.
